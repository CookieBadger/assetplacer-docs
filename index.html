The AssetPlacer is a plugin for the Godot game engine, that helps with level design by providing advanced asset browsing and placement tools. Get it now on itch.io!

If you like the plugin, and want to help me, please leave a review!

Not only can you save massive amounts of time using the AssetPlacer, it also helps you achieve the same things with fewer actions, thus you can get more done with less strain. Do you want to see the plugin in action? Watch this video, showing the creation of the scene above.
